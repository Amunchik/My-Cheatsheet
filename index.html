CSS Position Cheatsheet

CSS position property
Property name Value Description
position: static; The element is positioned according to the normal flow of the document.
position: relative; The element is positioned according to the normal flow of the document, and then offset relative to itself based on the values.
position: absolute; The element is removed from the normal document flow, and no space is created for the element in the page layout. It is positioned relative to its closest positioned ancestor, if any; otherwise, it is placed relative to the initial containing block.
position: fixed; The element is removed from the normal document flow, and no space is created for the element in the page layout. It is positioned relative to the initial containing block established by the viewport. Element will remain fixed to its position no matter where the user scrolls on the page.
position: sticky; The element is positioned according to the normal flow of the document, and then offset relative to its nearest scrolling ancestor and containing block. Keeps an element in the document flow as the user scrolls, but sticks to a specified position as the page is scrolled further.
Additional CSS properties
Name Property Values Desctiption
Offsets top; bottom; right; left; in pixels, ems, or percentages, among others Moves the element away from the mentioned side.
Z-index z-index: integer values, default value = 0 Controls how far back or how far forward an element should appear on the web page when elements overlap.
Display display: inline; block; inline-block; Dictates if it can share horizontal space with other elements.
Float float: right; left; Moving an element as far left or as far right as possible in the container. Floated elements must have a width specified.
Clear clear: right; left; both; none; Specifies how elements should behave when they bump into each other on the page. Mentioned in value side of the element will not touch any other element within the same containing element.
